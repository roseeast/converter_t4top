# Convert Top4Top

---

### Installation

### 1. Clone Repository
git clone https://github.com/roseeast/convert-top4top.git  

pip install -r requirements.txt

---

### 2. Install FFmpeg

Download FFmpeg:

https://www.gyan.dev/ffmpeg/builds/

Download file:

ffmpeg-release-essentials.zip

Open folder:

ffmpeg/bin

Add folder to Environment Variables PATH.

Example:

C:\ffmpeg\bin

Verify installation:

ffmpeg -version
